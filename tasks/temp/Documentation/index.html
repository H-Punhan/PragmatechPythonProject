Remove Header Sticky

Remove "header-sticky" class on Body tag.


<body class="header-sticky">

Change Images and custom Height for Slider ( Hero Section )

Please go to "hero-section" tag (id="hero-secion") and:

Change the height of Slider, ex: data=height="728"

Change the image link + the number of images, ex: data-number="2"


<section id="hero-section" data-height="0" data-number="3" data-image-1="assets/img/slider/slide1.jpg" data-image-2="assets/img/slider/slide2.jpg" data-image-3="assets/img/slider/slide3.jpg" data-effect="fade">

Contact Form

To edit the contact form to send mails to your email box, you can look for the contact-process.php. You need to edit yourname@yourdomain.com to your email.

contact-process.php ( includes/contact/contact-process.php )


define("WEBMASTER_EMAIL", 'yourname@yourdomain.com');
